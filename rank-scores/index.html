178. Rank Scores

2018年12月12日

计算出每个成绩的排名，要求如果分数相同那么排名也要一样。

这一题我是看了讨论区才学会的。

思路一

计算有多少比当前分数更高的分数。

思路二

和思路一类似的想法。

思路三

采用内连接，其中 t 表相当于是去重复后的分数，然后 s 表用来计算排名。

思路四

pre 是前一名的分数，如果当前分数和之前的分数不相同就返回 true，相等的话就返回 false。如果是 true，那么 rank 就会加 1，否则就是加 0。

我还没写过这么复杂的 SQL 语句。
